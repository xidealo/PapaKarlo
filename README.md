# PapaKarlo
This application allows you to order food from a cafe

### Release
You can download app from [Google Play](https://play.google.com/store/apps/details?id=com.bunbeuaty.papakarlo)

### Stack
- Kotlin
- Dagger2
- Kotlin Coroutines
- OkHttp + Retrofit
- Android Jetpack
- Firebase
- Coil
- MVVM
- Single Activity
